Improvements

Integral and Derivative Control

For both our line sensors and motor control we only have proportional control. More accurate movement can be possible if those are implmented as well.

Alternate Proximity Sensors

The HC-SR04 is good as a cheap alternative for a range finder but we have noticed that it triggers falsely quite often. Instead, it would yield better results if we used a TOF timer which would be more reliable.

Different Sensor Arrays

Currently, we are using 3 different sensors 1 5 array sensor and 2 1 channel array. It has issues espcially on the dotted lines as a fast adjustment onto the line could through off the dotted line and the sensors would miss the line and go off. If we had more sensors on the edges it could possibly catch itself even if that happens. However, if we had more precise line following logic and controls that would likely be unecessary as well.

Logic to Return to Start

The way we are approaching to return to start is to face in the y-direction and go to zero. Then face the x-direction and then go to zero in that direction. However, the issue is that with our IMU it doesn't perfectly face in the correct direction axis direction so traveling straight afterward would cause Romi to travel in a diagonal and stop at the wrong place. It would be much more efficient if a driving system can be implemented to directly change the wheel speeds to get both x and y values throughout its travel instead of trying to isolate the movement in the 2 axis.
