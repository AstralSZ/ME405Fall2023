Hardware

We are using the a pair of motors and encoders provided with the Romi Robot kit. In addition, the microcontroller we are using is the Nucleo-L476RG board along with the "Shoe of Brian" to be able to use Micropython to run the commands. Those are provided by the instructor of the class and is used for all other robots in the classroom. For the sensors, we are using an array of 5 channel line sensors posisitoned in the middle and 2 addition 1 channel sensors on the edges. 3D printed parts that can accomodate hardware are also used to attach the sensors onto the chassis of the robot. Our design also uses the IMU for more accurate positioning and dead reckoning which was also provided for the class and used for a seperate Lab assignment. For navigation around the wall, we are using the HC-SR04 ultrasonic sensor.
